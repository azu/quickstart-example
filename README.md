# QuickStart Example

Example project of [QuickStart](http://spotify.github.io/quickstart/ "QuickStart").

## Installation

```sh
npm install
# develop
npm run quickstart
npm run watch # == broserify
# build
npm run build-q # build with quickstart
npm run build-b # build with browserify
```
